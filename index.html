What are clients

(Back)

Used in home and corporate networks, a client is any computer hardware or software device that requests acces to a service provided by a server. Clients are typically seen as the requesting program or user in a client-server architecture. Client end-user devices typically include desktop computers, laptops and smartphones.

In a computer network, a client in a client-server model is what requests a service or resource from a server. The server can be located on or off premisses.

Understanding the difference between ordered and unordered lists

(Back)

An unordered list (ul) is used to create a list of items in no particular order, that is, the order of the items is not relevant. By default, items in this list will be marked with bullets. Whereas an ordered list (ol) is used to create a list of items in a specific order.
